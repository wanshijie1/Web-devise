示例文章

这是一篇示例文章，旨在展示如何使用 HTML 创建简单的文章页面。

文章段落

文章通常被分成多个段落，每个段落包含一个主题或想法。段落通常由一两句话组成，以便读者可以快速了解其内容。

标题

标题通常用于在文章中引导读者，同时将不同的部分分隔开来。标题的层次结构通常使用 h1 到 h6 六个级别，以便读者可以了解标题之间的层次结构。

引用

引用可以用于引用其他人的话或观点，或者强调文章中的某些内容。在 HTML 中，可以使用 blockquote 元素来创建引用。

生活是一种不断探索的过程，无论你做什么，总是会学到新的东西。

——约翰·戴维森·洛克菲勒

列表

列表用于列举一些事物，或者按照一定的顺序呈现信息。在 HTML 中，可以使用 ul 元素创建无序列表，或者使用 ol 元素创建有序列表。

  • 苹果
  • 香蕉
  • 橙子

图片

图片可以帮助读者更好地理解文章中的内容，或者为文章增加视觉效果。在 HTML 中，可以使用 img 元素来插入图片。

链接

链接可以将文章中的不同部分或不同页面连接起来，或者将读者带到其他有用的资源。在 HTML 中，可以使用 a 元素来创建链接。
